🛠 План розробки проєкту: Математичний чат-бот
1. Концепція продукту

Тип додатку: Чат-бот
Основна ідея:
Чат-бот, який приймає текстові запити користувача та виконує математичні розрахунки (від простих арифметичних до більш складних виразів), повертаючи результат у зрозумілому вигляді.

Яку проблему вирішує:

Швидкий підрахунок без калькулятора
Обробка текстових математичних запитів (“скільки буде 2+2*5”)
Автоматизація простих обчислень у чаті
2. Основний функціонал (Features)

✅ Приймати текст від користувача

Ввід через чат (CLI, Telegram, веб-інтерфейс)

✅ Обробляти математичні вирази

Парсинг тексту
Підтримка операторів: + - * /
Дужки ( )

✅ Виконувати розрахунок

Обчислення виразу
Обробка помилок (наприклад, ділення на нуль)

✅ Виводити результат

Текстовий формат
Повідомлення про помилки

🔹 Додатково (рекомендовано):

Історія обчислень
Підтримка складніших функцій (sin, log, sqrt)
Розуміння природної мови (“додай 5 до 10”)
3. Структура даних
Назва змінної	Тип даних	Для чого потрібна
user_input	str	Ввід тексту користувача
parsed_expr	str	Очищений/оброблений вираз
result	float	Результат обчислення
is_active	bool	Контроль роботи чат-бота
history	list	Збереження попередніх обчислень
Схема:
[Старт]
   ↓
[Запуск бота]
   ↓
[Очікування вводу користувача]
   ↓
[Отримати user_input]
   ↓
{user_input пустий?}
   ├── Так → [Показати "Введіть вираз"] → назад до очікування
   └── Ні
        ↓
[Обробка тексту (очистка)]
        ↓
[Парсинг виразу]
        ↓
{Вираз коректний?}
   ├── Ні → [Показати помилку] → назад до очікування
   └── Так
        
[Обчислення результату]
        ↓
{Помилка під час обчислення?}
   ├── Так → [Показати помилку] → назад до очікування
   └── Ні
        ↓
[Вивести результат]
        ↓
[Зберегти в history]
        ↓
{Користувач хоче вийти?}
   ├── Так → [Завершити роботу]
   └── Ні → назад до очікування
